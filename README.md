# Site Relamping LED - Corse & DOM-TOM

Site vitrine React (Vite + TailwindCSS).

## 🚀 Développement local
```bash
npm install
npm run dev
```

## 📦 Build de production
```bash
npm run build
```

## 🌐 Déploiement
Déployez facilement sur Vercel, Netlify ou tout hébergeur supportant les apps Vite/React.
